# Python Control System

Sistema de Controle de Estoque desenvolvido em Python, com foco em organização,
aprendizado de lógica de programação e boas práticas.

# Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Atualização de estoque
- Remoção de produtos
- Registrar e controlar vendas
- Controle básico via terminal

# Tecnologias utilizadas
- Python
